## 期末考用
#### Q1

$f(x) = e^{2x}*sin(3x) $:mathjax    

$$ \frac {d}{dx}f(x) = 2*e^{2x}*sin(3x) + e^{2x}*3*cos(3x) $$    

$$\int f(x) dx. = \frac {1}{13} * e^{2x} * (2*sin(3x) - 3*cos(3x)) $$    

\#1
用3點、5點差分法解 $ \frac {df}{dx} $ ，h(間距)自訂，誤差要壓在 $ 10^{-4}$ 以內。求h?    

\#2
用任一方式 $ \int_0^2 f(x)dx. $ ，h(間距)自訂，誤差要壓在 $ 10^{-4}$ 以內。求h?
